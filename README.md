# STM32MP157

*** stm32mp157 project ***

- Development Platform：STM32MP157A-DK1 (官方 demo 板)

- Host PC：Ubuntu22.04

tools 文件夹包含交叉编译工具，使用前先设置环境变量

``` bash
export PATH=$PATH:/home/$USER/STM32MP157/tools/arm-linux-gcc-10.3/bin
```
